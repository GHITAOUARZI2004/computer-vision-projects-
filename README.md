## 📌 Overview

This project implements a complete computer vision pipeline using YOLOv8 and DeepSORT.

It detects people in a video, assigns unique IDs to each individual, and counts the total number of people.



## 🚀 Features

* Real-time object detection (YOLOv8)
* Multi-object tracking (DeepSORT)
* Unique ID assignment for each person
* People counting system



## 🛠️ Technologies Used

* Python
* OpenCV
* YOLOv8 (Ultralytics)
* DeepSORT



## 📂 Project Structure

* `main.ipynb` → Colab notebook
* `results/` → output video/images



## 🎯 Output

* Bounding boxes around people
* Unique ID per person
* Total count displayed



## 🔮 Future Improvements

* Line crossing detection (IN / OUT)
* Heatmap visualization
* Trajectory tracking
